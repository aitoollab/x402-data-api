| 时间 (UTC) | 任务 | 频率 | 说明 |
|-----------|------|------|------|
| 00:00 | 生态扫描 | 每日 | 检查 x402scan/x402atlas 新 API |
| 06:00 | 竞品分析 | 每日 | 分析热门 API 定价、功能 |
| 12:00 | 收益报告 | 每日 | 统计 USDC 收入、调用量 |

## 决策规则

### 新 API 开发决策

```python
def should_develop(api_idea):
    score = 0

    # 竞争度评分 (0-40分)
    competitors = count_competitors(api_idea.category)
    if competitors == 0:
        score += 40  # 无竞争
    elif competitors <= 3:
        score += 30  # 低竞争
    elif competitors <= 10:
        score += 15  # 中竞争
    else:
        score += 5   # 高竞争

    # 需求量评分 (0-30分)
    demand = estimate_demand(api_idea.category)
    if demand == "high":
        score += 30
    elif demand == "medium":
        score += 20
    else:
        score += 10

    # 实施难度评分 (0-20分)
    difficulty = assess_difficulty(api_idea)
    if difficulty == "easy":
        score += 20
    elif difficulty == "medium":
        score += 10
    else:
        score += 5

    # 数据源可用性 (0-10分)
    if has_free_data_source(api_idea):
        score += 10
    else:
        score += 3

    return score >= 60  # 60分以上开发
```

### 价格调整规则

```yaml
提价条件:
  - 调用量 > 1000/天 且错误率 < 1%
  - 无直接竞争对手
  - 用户反馈积极

降价条件:
  - 调用量 < 100/天 超过7天
  - 有新竞争对手且价格更低
  - 错误率 > 5%

价格区间:
  - 基础数据: $0.01-0.02
  - 聚合数据: $0.03-0.05
  - 分析/计算: $0.05-0.10
  - 独家数据: $0.10-0.20
```
